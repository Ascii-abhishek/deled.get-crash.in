Section 2: Short Answer Questions (5 Questions)

1. Define Microorganism and write one classroom or exam point.

Microorganism: Living organism too small to see with naked eye. Seen with microscope.

2. Define Bacteria and write one classroom or exam point.

Bacteria: Single-celled microorganisms found everywhere. Useful and harmful both.

3. Define Virus and write one classroom or exam point.

Virus: Tiny infectious agent that multiplies inside living cells. Causes cold, flu, measles etc.

4. Define Fungi and write one classroom or exam point.

Fungi: Organisms like mould, yeast, mushroom. Yeast helps fermentation.

5. Define Protozoa and write one classroom or exam point.

Protozoa: Single-celled animal-like microorganisms. Some cause malaria, amoebiasis.
