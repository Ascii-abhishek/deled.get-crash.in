Core Points

  • Hindi evaluation should cover all language skills, not only grammar exercises.
  • Questions should match learning outcomes and class level.
  • Use oral, written, performance, portfolio, project, and observation tools.
  • Feedback should be immediate, clear, respectful, and improvement-oriented.
  • A good paper includes recall, understanding, application, expression, and creativity.
  • Evaluation becomes useful when it informs remedial teaching and enrichment.

Classroom Connection

Use a simple rubric for reading aloud: pronunciation, pause, fluency, meaning, and confidence.

One-minute Revision

  • Evaluation: Judging learning progress using evidence and criteria.
  • Assessment: Collecting information about learning through tasks and observation.
  • Learning outcome: A clear statement of what the learner should know or do.
  • Formative evaluation: Ongoing evaluation used to improve learning during teaching.
  • Summative evaluation: Evaluation at the end of a unit, term, or course.
  • Rubric: A scoring guide with levels and criteria.
