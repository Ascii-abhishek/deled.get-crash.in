Core Points

  • A set must be well-defined; vague collections are not sets in mathematics.
  • Use capital letters for sets and small letters for elements.
  • Roster method is useful for small sets; set-builder method is useful for large or patterned sets.
  • Empty set is written as {} or phi and has no element.
  • A Venn diagram shows relations between sets visually.
  • Use classroom collections: vowels, odd numbers, shapes, local fruits, or students who like two games.
