Section 2: Short Answer Questions (10 Questions)

1. Define Ratio with an example.

Comparison of two quantities of the same kind by division, written a:b or a/b.

2. Define Base with an example.

The number or expression repeatedly multiplied.

3. Define Principal with an example.

Original money borrowed or invested.

4. Define Bank with an example.

Institution that accepts deposits, lends money, and supports payments.

5. Define Logarithm with an example.

Power to which a base must be raised to get a number.

Section 3: Long Answer Questions (5 Questions)

1. Explain the main concepts of Ratio and Proportion with examples.

A good answer should include definition, example, process, and classroom use.

  • Compare only like quantities; convert units before forming a ratio.
  • Simplify a ratio by dividing both terms by their HCF.
  • Same units first, HCF next, cross product last.
  • Give groups of bottle caps in two colours. Learners form ratios, simplify them, then predict missing caps using proportion before writing symbolic solutions.
2. Explain the main concepts of Indices with examples.

A good answer should include definition, example, process, and classroom use.

  • Indices shorten repeated multiplication and make calculation compact.
  • Apply addition of powers only when bases are the same and terms are multiplied.
  • Multiply same base: add powers. Divide same base: subtract powers.
  • Use folded paper or doubling charts to show powers of 2, then move to symbolic rules and error spotting.
3. Explain the main concepts of Simple and Compound Interest with examples.

A good answer should include definition, example, process, and classroom use.

  • Simple interest formula: SI = (P x R x T) / 100.
  • Amount in simple interest: A = P + SI.
  • Simple: interest stays on P. Compound: interest joins P.
  • Run a class savings-bank activity: groups calculate simple interest first, then compound interest for the same data and discuss why amounts differ.
4. Explain the main concepts of Knowledge of Banking with examples.

A good answer should include definition, example, process, and classroom use.

  • Banking mathematics uses percentage, interest, balance, deposits, withdrawals, and service charges.
  • Savings accounts encourage financial discipline; current accounts suit frequent business transactions.
  • Banking math = old balance + deposits - withdrawals.
  • Set up a mock bank counter where learners fill deposit slips, update a passbook table, and calculate new balances.
5. Explain the main concepts of Logarithm with examples.

A good answer should include definition, example, process, and classroom use.

  • Logarithm answers: what power gives this number?
  • If 10^3 = 1000, then log10 1000 = 3.
  • Logarithm is the question mark on the power.
  • Use a table of powers of 10 and ask learners to read it both ways: exponential statement and logarithmic statement.
