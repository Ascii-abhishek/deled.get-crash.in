Section 2: Short Answer Questions (5 Questions)

1. Write a short note on Data.

Data means Facts or numbers collected for study. Example: Marks, height, rainfall. In exams, write its definition, use, and one classroom connection.

2. Write a short note on Tabulation.

Tabulation means Arranging data in rows and columns. Example: Frequency table. In exams, write its definition, use, and one classroom connection.

3. Write a short note on Bar graph.

Bar graph means Graph using equal-width bars. Example: Class-wise attendance. In exams, write its definition, use, and one classroom connection.

4. Write a short note on Mean.

Mean means Sum of values divided by number of values. Example: Average marks. In exams, write its definition, use, and one classroom connection.

5. Write a short note on Median.

Median means Middle value after arranging data. Example: Central score. In exams, write its definition, use, and one classroom connection.

Section 3: Long Answer Questions (4 Questions)

1. Define statistics and data.
  • Statistics helps collect, organise, present, analyse and interpret data.
  • Raw data should be arranged before drawing conclusions.
  • Tables make comparison quick; graphs make patterns visible.
  • Mean is affected by extreme values; median is useful for ordered data with extremes.
2. Differentiate mean, median and mode.
  • Raw data should be arranged before drawing conclusions.
  • Tables make comparison quick; graphs make patterns visible.
  • Mean is affected by extreme values; median is useful for ordered data with extremes.
  • Mode is the most frequent value and helps identify common preference.
3. Explain tabulation.
  • Mean: Uses every value (Affected by very high or low value)
  • Median: Middle value (Less affected by extremes)
  • Mode: Most frequent value (Useful for common size or preference)
4. When is bar graph useful?
  • Collect class attendance for a week and draw a bar graph.
  • Use marks of five students to calculate mean, median and mode.
