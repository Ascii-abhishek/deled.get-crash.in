Section 2: Short Answer Questions (5 Questions)

1. Write a short note on Stimulus.

Stimulus means Event that starts a response. Example: teacher praise. In exams, write its definition, use, and one classroom connection.

2. Write a short note on Response.

Response means Behaviour shown after stimulus. Example: participation. In exams, write its definition, use, and one classroom connection.

3. Write a short note on Reinforcement.

Reinforcement means Consequence increasing desired behaviour. Example: appreciation. In exams, write its definition, use, and one classroom connection.

4. Write a short note on Punishment.

Punishment means Consequence reducing behaviour but may create fear. Example: scolding. In exams, write its definition, use, and one classroom connection.

5. Write a short note on Behaviour modification.

Behaviour modification means Planned change in behaviour using cues and reinforcement. Example: class routine. In exams, write its definition, use, and one classroom connection.
