Key Points

  • Original writing grows from speaking, drawing, experience sharing and word collection.
  • Familiar topics reduce fear because children already have ideas and vocabulary.
  • Letter writing must include format, purpose, clear language and polite closing.
  • Correction should cover idea order, sentence clarity, spelling, punctuation and presentation.
  • Teacher should value children's own examples before polishing grammar.

Answer Writing Frame

  1. Write a clear definition of the key term.
  2. Add one local or textbook example.
  3. Put reason, steps or difference in points/table.
  4. End with one short primary-class use.

Classroom Connection

Use a five-step writing cycle: talk, list words, make outline, write first draft, revise with peer checklist.

Memory Hook

Term -> Example -> Reason/steps -> Classroom activity -> Revision

Common Exam Lines

  • In Original Writing on Familiar Topics, define the term, add one example, and connect it with classroom use.
  • For difference questions, write basis, feature, and example in separate lines.
  • For primary teaching, add local examples, pictures, cards, activity and oral practice.
