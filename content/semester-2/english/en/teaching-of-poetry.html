Key Points

  • At primary level, poetry teaching should create enjoyment, rhythm and confidence.
  • Teacher's model recitation is central; children imitate tone, pause and action.
  • Meaning should be explained through pictures, action and simple questions, not long translation.
  • Poems improve pronunciation, listening, memory and vocabulary.
  • Assessment can include recitation, rhyme matching, action, drawing and one-line meaning.

Answer Writing Frame

  1. Write a clear definition of the key term.
  2. Add one local or textbook example.
  3. Put reason, steps or difference in points/table.
  4. End with one short primary-class use.

Classroom Connection

Use listen-repeat-act: teacher recites, children repeat line by line, add actions, clap rhyme words, and draw one image from the poem.
