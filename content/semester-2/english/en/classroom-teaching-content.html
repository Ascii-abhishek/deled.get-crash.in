Key Points

  • Content should move from known objects to new English words.
  • Greetings and classroom phrases create a real need to use English daily.
  • Alphabet teaching should connect letter shape, sound, word and picture.
  • Number names need oral counting, matching numerals and word cards.
  • Reading simple sentences should follow oral practice and vocabulary familiarity.

Answer Writing Frame

  1. Write a clear definition of the key term.
  2. Add one local or textbook example.
  3. Put reason, steps or difference in points/table.
  4. End with one short primary-class use.

Classroom Connection

Make a classroom object walk: children pick an object card, say its English name, make one sentence, and place it under the correct first letter.

Memory Hook

Term -> Example -> Reason/steps -> Classroom activity -> Revision

One-minute Revision

  • Greeting: Polite words used when meeting
  • Alphabet: Set of English letters
  • Letter recognition: Identifying letters by shape and sound
  • Vocabulary: Words known and used by learner
  • Number names: Writing numbers in words
  • Oral practice: Speaking practice before writing

Common Exam Lines

  • In Classroom Teaching: Content, define the term, add one example, and connect it with classroom use.
  • For difference questions, write basis, feature, and example in separate lines.
  • For primary teaching, add local examples, pictures, cards, activity and oral practice.
