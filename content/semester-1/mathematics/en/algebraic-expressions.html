Algebraic Expressions

This chapter introduces variables, constants, coefficients and terms, then revises addition and subtraction of like algebraic expressions.

Exam Snapshot

TermMeaningExample
VariableA symbol whose value can change.x, y, n.
ConstantA fixed number.7 in 3x + 7.
CoefficientNumber multiplied by a variable.5 in 5a.
TermA part separated by + or - signs.3x, -2y, 8.
ExpressionCombination of variables, constants and operations.4x + 3.

Key Points

  • Algebra uses letters to represent unknown or changing numbers.
  • An expression does not have an equality sign; an equation has an equality sign.
  • Like terms have the same variable part and same powers, such as 3x and 8x.
  • Unlike terms have different variable parts, such as 3x and 3y.
  • Only like terms are added or subtracted directly.

Operations

  • Add coefficients of like terms: 3x + 5x = 8x.
  • Subtract coefficients of like terms: 9a - 4a = 5a.
  • Keep unlike terms as they are: 4x + 3y cannot become 7xy.
  • Remove brackets carefully: 2a + (3a - 5) = 5a - 5.
  • Use substitution to find value: if x = 2, then 3x + 4 = 10.

Classroom Connection

Use matchsticks for patterns: one square uses 4 sticks, two separate squares use 8, and n squares use 4n. This helps children see a variable as a changing number.

One-minute Revision

  • Variable changes; constant stays fixed.
  • Coefficient is the numerical factor of a term.
  • Expression has no equality sign.
  • Like terms have the same variable part.
  • Add or subtract coefficients, not variable names.

Common Exam Lines

  • Algebra is generalized arithmetic using symbols for numbers.
  • Like terms can be combined because their variable parts are the same.
  • Algebraic expressions should be taught through patterns, tables and word situations.
